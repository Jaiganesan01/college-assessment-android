# College Assessment Management System (Android)

This project is an academic prototype of a College Assessment Management System designed for large-scale student usage.

## Features
- Role-based login for students and faculty
- Secure authentication system
- Time-limited online assessments
- Automatic submission on exit or timeout
- Faculty dashboard for assessment creation and result viewing
- Cloud-ready and scalable architecture

## Tech Stack
- Android (Java/Kotlin)
- Open-source backend architecture
- GitHub for version control

## Note
This repository contains the complete source code. The APK can be generated using Android Studio or cloud CI tools.
